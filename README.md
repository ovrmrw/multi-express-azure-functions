# multi-server-azure-functions
(experimental)

---

## Setup
```
$ npm install
```

## Run local server
```
$ npm install -g azure-functions-cli
$ npm run local
```

(atention) Make sure to be installed Visual Studio (NOT VSCode) before using `azure-functions-cli`.

## Unit Test
```
$ npm test
```

## Deploy
```
$ npm run deploy
```
